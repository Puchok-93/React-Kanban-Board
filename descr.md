Добавление карточки

в App 

1. Берем предыдущее состояние prevColumns.
2. Проходим по всем колонкам.
3. Если колонка не backlog, возвращаем её без изменений.
4. Если это backlog, создаем новую колонку, в которой:
- копируем старые свойства (...column);
- создаем новый массив задач;
- добавляем новую задачу в конец.

в AddTask

AddTask отвечает только за форму (ввод текста, отправка, очистка поля).
App отвечает за изменение состояния доски.
Taskboard и Column только отображают данные.
